実行環境は構築してるはずだからそのまま実行してみてね
実行はsrc/index.jsです

階層については
道路か河川かのフォルダーの下に各国交省事務所の階層で成り立ってます
なのでその下からすべてが実行するフォルダー類です
なので中身はその下からすべてはすべてどこのフォルダーはすべて一緒

//ここから環境構築のメモ

1.  >npm init -y
2.  >npm install typescript
3.  npx tsc --init
4.  tsconfig.jsonの　"sourceMap": trueをオンにする　（//を消す）
5.  ターミナル☞既定のビルドタスクの構成を押す
6.  tsc:ビルドー tsconfig.jsonを押す
7.  実行☞構成の追加をクリック
8.  Nose.jsを選ぶ
9.  "program": "src\\index.js"を書く（元もあるところに記載して！下の方にあるから）
10. src/index.jsを実行

これは各サイト(環境)ごとに違います
load/◯◯(国道事務所ごとの名前)/~  もしくは   river/◯◯(河川事務所ごとの名前)/~ここから環境です
